Vamos a elaborar nuestra primera web con XHTML intentando dar formato básico al texto y evitando introducir ningún texto que no haya sido previamente preformateado. Para ello vamos a utilizar las etiquetas básicas que XHTML nos proporciona para trabajo con texto.

Es importante recordar que toda etiqueta que abramos hemos de cerrarla, siguiendo el estándar predefinido en XHTML. Esta será nuestra forma habitual de trabajo, a la que hemos de ceñirnos para evitar errores o futuros problemas.
